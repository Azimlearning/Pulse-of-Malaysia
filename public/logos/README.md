# Logo Placeholders — Action Required

Drop your competition logo files here before submission.

## Required Files

| Filename | Description | Source |
|----------|-------------|--------|
| `ukm.png` | Universiti Kebangsaan Malaysia official logo | UKM brand portal |
| `ukm-watan.png` | UKM Watan official logo | Competition briefing slides |
| `data-challenge.png` | Data Challenge 5.0 official logo | Competition briefing slides |
| `sdg3.png` | SDG 3 — Good Health and Well-Being icon | https://www.un.org/sustainabledevelopment/news/communications-material/ |
| `sdg8.png` | SDG 8 — Decent Work and Economic Growth icon | https://www.un.org/sustainabledevelopment/news/communications-material/ |

## After Dropping Files

Open `src/components/LogoBar/logos.config.ts` and set `available: true` for each logo you've added.
The component will automatically switch from the dashed placeholder to the real image.

## Recommended Dimensions
- UKM, UKM Watan, SDG icons: 72×72px or larger (displayed at 36×36px)
- Data Challenge 5.0: 160×72px or larger (displayed at 80×36px)
- Format: PNG with transparent background preferred
